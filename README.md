# Are.na New Tab Curator

Chrome extension to show random blocks from your Are.na channel on new tabs.

## Setup
1. Get Are.na Personal Access Token: are.na/settings/apps.
2. Get Channel ID from channel URL.
3. Load extension in Chrome: chrome://extensions > Load unpacked > select folder.
4. Go to extension options, enter token and channel ID.

## Development
Clone repo, edit files, reload extension.
